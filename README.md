# islamnet
Islamnet AI Chatbot, an AI companion that tells you answers to Islamic questions using rulings.
 IslamNet | The Digital Islamic Suite

IslamNet is a high-performance, all-in-one Islamic dashboard designed with a "bulky" pure black glassmorphism aesthetic. Built for the modern user, it combines essential spiritual tools with an elegant, space-inspired interface featuring interactive glowing stars and a seamless Single Page Application (SPA) architecture.
 Features

     AI Companion: A built-in intelligent chat interface for exploring Islamic knowledge and answering queries instantly.

     Smart Prayer Times: A Google-inspired prayer module with location-based detection and multiple juristic calculation methods (ISNA, MWL, Umm al-Qura).

     Interactive Hijri Calendar: A custom-built lunar calendar with real-time month and year navigation (Umm al-Qura standard).

     Daily Inspiration: Automated daily refresh of Quranic Ayahs (Arabic/English) and Sahih Hadith.

     Integrated Knowledge Base: Full-screen, maximized viewports for Quran.com and Sunnah.com, allowing for a deep-reading experience without leaving the site.

     Audio Recitations: (In Progress) A dedicated module for world-renowned reciters.

 Design Philosophy

    OLED Pure Black: Optimized for high-contrast viewing and battery efficiency.

    Glassmorphism UI: Deep background blurs and translucent borders create a premium "glass" feel.

    Interactive Atmosphere: A dynamic, physics-based starry background where stars pulse and glow based on unique timing variables.

    Apple-Style Navigation: A floating bottom pill-bar for effortless switching between tools.

 Built With

    HTML5 & CSS3: Custom styling using the Montserrat font family.

    JavaScript (Vanilla): Custom logic for the Hijri calendar, API fetching, and view management.

    Aladhan API: For precise global prayer timings.

    AlQuran Cloud API: For random ayah generation.

    Vercel/GitHub Pages: Optimized for fast, global deployment.

 How to use

    Prayers: Enter your city and select your preferred calculation method.

    Calendar: Use the arrows to browse through the Hijri months and years.

    Chat: Ask questions directly to the AI to get scholarly insights.
